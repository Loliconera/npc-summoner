# npc-summoner

Summon any NPC (merchant, bank, broker) from anywhere.

## Commands
Toolbox(/8) | Command description
--- | ---
**broker** | Trade Broker
**bank** | Bank
**gbank** | Guild Bank
**pbank** | Personal Storage
**store** | Merchant
**sstore** | Specialty Store
**vg** | Vanguard Initiative Credit Merchant
